# Teaching Notes

## Diagram & Graph Requirements
- **Mermaid only** — all diagrams and graphs in lessons MUST be rendered via [Mermaid.js](https://mermaid.js.org/), included from CDN in each HTML lesson. Never use ASCII/text-art diagrams.
- Include `<script type="module">` Mermaid initialisation in every lesson that contains a diagram.
- Use `graph TD` for hierarchies, `sequenceDiagram` for flows, `classDiagram` for relationships.

## Your Preferences
- **Primary Goal**: Shape threat identification mindset (how to *think* about threats)
- **Teaching Context**: AI-assisted threat modelling for security reviews
- **Style**: You want to develop judgment, not just memorize frameworks
- **Integration**: Learning to use AI effectively as a partner tool

## Reminders for Future Lessons
- Keep lessons focused on mindset and reasoning patterns
- Use AI examples showing both where it's useful and where it can fail
- Ground everything in real security review scenarios
- Connect each lesson back to "what would I look for here?"
